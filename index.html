What is CSS?

CSS(Cascading Style Sheet) is a stylesheet language that is used to selectively style HTML elements.
To link the CSS to an HTML document, a link tag is used using the nameofdocument.css as the href attribute of the link.

Selector

This is the HTML element to be styled e.g p for paragraph.

Declaration

This specifies the property of the element to be styled

Properties

These are the ways in which HTML elements can be styled.Some of these properties include: color, font,width

Some other parts of the syntax to take note of are as follows:

  • Apart from the selector,all other declarations and properties must be within curly brackets{}
  • There must be a colon :within every declaration
  • A semicolon ; must be used to seperate the declarations
  • Multiple elements can be selected to apply the same style to them. The elements have to be seperated by commas

The list below explains types of selsctors in CSS:

  1. Element Selector:This is the HTML tag to be styled.For instance p is the element selector for paragraphs
  2. ID selector:An id value should be unique i.e no two elements should have the same id. Example of styling an element with id #idvalue
  3. Class selector:This is used to style elements within the same class. The syntax is .classname
  4. Attribute selector":The element(s) on the page with the specified attribute.For example: img[src]
  5. Pseudo-class selector:The specified element(s), but only when in the specified state. For example hovera:hoverwhen the cover hovers on the link, the declarations made can be seen e.g underline

The property font-family refers to the font(s) you want to use for text.Font-size is used to set a value for the size of the texts.The elements inherit the font properties of the HTML body unless specific font properties are assigned to certain elements.

CSS layout is mostly based on the box model i.e properties like size,position e.t.c. are arranged in forms of boxes.Each box on the page can be represented by the image below:

  • Padding:This is the space around the content.In the image above,it is the space around the paragraph text.
  • Border:This is the solid line outside the padding.
  • Margin:This is the space around the outside of the border.

Some other properties to take note of are:

  • Width of an element
  • Background color, the color behind the contents
  • Color, the color of the elements contenets
  • Textshadow, sets a shadow on the contents
  • Display, sets the display mode of an element
